# Examples

Sample files for trying the extension. Right-click any of them in the Explorer
and choose **AI Review: Open Review Panel**.

- `design-doc.md` — Markdown with a `mermaid` diagram. Shows preview rendering
  and element/line comments mapped back to source lines.
- `landing.html` — a small landing page. Shows live HTML preview + clicking
  elements to comment.
- `config.yaml` — source-only review; line comments capture the YAML data path.
